# Dicke State Generation via Generalized Parity Measurement

This repository contains a MATLAB script for simulating the **measurement-based Dicke-state generation protocol** proposed in:

> Chen-yi Zhang and Jun Jing, *Efficient nonclassical state preparation via generalized parity measurement*, **Physical Review A 113, 022420 (2026)**.

The code corresponds to the **Dicke-state generation** part of the paper, where a large spin ensemble is driven toward a target Dicke state by repeated nonunitary filtering steps induced by evolution and projective measurement.

## Overview

The protocol works in the permutation-symmetric subspace of a spin ensemble with total spin

$$\[
J = M/2,
\]$$

where $\(M\)$ is the number of spin-1/2 particles.

The script:

- constructs the collective spin operators in the Dicke basis,
- prepares an initial spin-coherent-like state,
- defines a target Dicke state $\(|J,m_t\rangle\)$,
- applies a sequence of diagonal nonunitary filtering operators,
- tracks the fidelity, success probability, and quantum Fisher information (QFI).

The numerical logic follows the Dicke-state section of the paper, especially the discussion around the spin-star model and the effective generalized parity measurement used to prepare the center Dicke state.

## Physical meaning of the script

The Hilbert space is the Dicke basis

$$\[
|J,-J\rangle, |J,-J+1\rangle, \dots, |J,J\rangle,
\]$$

with dimension

$$\[
2J+1.
\]$$

The target state is chosen as

$$\[
|J,m_t\rangle,
\]$$

and in the current script the default choice is the **center Dicke state**

$$\[
|J,0\rangle.
\]$$

The protocol applies a sequence of approximate generalized-parity filters. After each successful round, the state is renormalized and becomes progressively concentrated near the target Dicke state. This is the same mechanism described in the paper for constructing an effective GPM in the spin-ensemble setting. fileciteturn0file0L321-L349 fileciteturn0file0L350-L386

## Output quantities

The script stores three main diagnostics:

### 1. Fidelity

```matlab
Pn(ii)
```

This is the fidelity with the target Dicke state after round `ii`:

\[
P_n^{(i)} = |\langle \text{target}|\psi_i\rangle|^2.
\]

### 2. Success probability

```matlab
Pg(ii+1)
```

This is the cumulative probability that all measurements up to that round return the desired outcome.

### 3. Quantum Fisher information

```matlab
Fisher(ii)
```

This is computed as

\[
F_Q = 4\left(\langle J_x^2\rangle - \langle J_x\rangle^2\right),
\]

which is the QFI for phase estimation generated by \(J_x\). The paper uses this quantity to assess the metrological usefulness of the generated Dicke states. fileciteturn0file0L387-L412

## How to run

Open MATLAB and run:

```matlab
run('dicke_state_generation_commented.m')
```

The script will generate a figure with two panels:

- **Top panel:** fidelity versus round number
- **Bottom panel:** cumulative success probability versus round number

### Number of rounds

### System size

```matlab
J = 50;
```

This corresponds to a spin ensemble of

```matlab

```

spin-1/2 particles in the symmetric subspace.

### Target Dicke state

```matlab
mt = 0;
```

This selects the target \(|J,m_t\rangle\). The default is the center Dicke state.

## Structure of the code

### 1. Collective spin operators

The script builds:

- `Jm` : lowering operator \(J_-\)
- `Jp` : raising operator \(J_+\)
- `Jx` : collective spin operator \(J_x\)
- `Jy` : collective spin operator \(J_y\)

These are represented in the Dicke basis.

### 2. Initial state

The script starts from the fully polarized Dicke state and rotates it by

\[
e^{iJ_y\pi/2},
\]

to prepare a spin-coherent-like state centered around the target region.

### 3. Nonunitary filtering

Two diagonal coefficient functions are defined:

- `Vlee_dicke`
- `Vlgg_dicke`

These implement the round-dependent filtering factors. The first round is treated separately, and later rounds use a halved-time sequence, consistent with the generalized-parity strategy in the paper. fileciteturn0file0L350-L386

### 4. Iteration loop

At each round:

1. construct the diagonal filtering operator,
2. apply it to the current state,
3. renormalize the state,
4. record fidelity,
5. update success probability,
6. compute QFI.

## Relation to the paper

The paper shows that for large spin ensembles, the Dicke-state preparation can be achieved with a number of rounds that scales approximately as

\[
N_{\text{Dicke}}(M) \sim \log_2(\sqrt{M}),
\]

and gives numerical examples with high fidelity for large \(M\). fileciteturn0file0L373-L386 fileciteturn0file0L387-L412

It also shows that the generated state can already exhibit near-Heisenberg QFI scaling after only a few rounds. fileciteturn0file0L398-L412

## Notes

1. The script works entirely in the **symmetric Dicke subspace**, so it is much more efficient than simulating the full \(2^M\)-dimensional Hilbert space.
2. The variable `Pt_Dicke` is present in the original code but is not used in the current version.
3. The present script focuses on the **ideal conditional evolution** and does not include decoherence or full trajectory sampling.
4. The code is written in a compact research style; the commented version is intended to make each step easier to reuse and modify.
